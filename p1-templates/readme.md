## Set up
Recommend python 3.7 or later to install tensorflow2 and keras

```
# Prerequisite
$ conda create -n 703hw2
$ conda install python=3.8.5
$ pip install tensorflow
$ pip install keras
$ pip install matplotlib
$ pip install gym
```

## Compile
```
$ python BCDAGGER.py
```
